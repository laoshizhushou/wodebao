# 老师的小能手 - 网站部署教程

## 准备工作

你需要：
1. 一个邮箱（用来注册GitHub）
2. 一台电脑（Windows/Mac都可以）
3. 你的微信二维码图片（用来让老师加你微信）

---

## 第一步：注册GitHub账号（如已有可跳过）

1. 打开浏览器，访问：https://github.com
2. 点击右上角 "Sign up"
3. 输入你的邮箱，点击 "Continue"
4. 设置密码，点击 "Continue"
5. 输入用户名（比如：teacher-helper），点击 "Continue"
6. 选择 "y" 或 "n"（都可以），点击 "Continue"
7. 去邮箱查收验证邮件，点击验证链接
8. 完成注册！

---

## 第二步：准备微信二维码

你需要把自己的微信二维码放到网站上：

**方法1：截图保存（最简单）**
1. 打开手机微信
2. 点击 "我" → 头像旁边的小箭头
3. 点击 "二维码名片"
4. 截图保存到电脑，重命名为 `wechat-qr.png`
5. 放到本文件夹（老师的小能手）中

**方法2：生成二维码图片**
1. 打开草料二维码网站：https://cli.im
2. 选择 "微信" 标签
3. 输入你的微信号
4. 点击 "生成二维码"
5. 下载二维码图片，重命名为 `wechat-qr.png`
6. 放到本文件夹中

---

## 第三步：修改网站（添加你的二维码）

1. 用记事本打开 `index.html` 文件
   - 右键 index.html → 打开方式 → 记事本

2. 用 Ctrl+F 搜索 `qr-placeholder`

3. 找到这一段代码：
```html
<div class="qr-placeholder">
    微信二维码<br>（请替换为真实二维码）
</div>
```

4. 把它替换成：
```html
<img src="wechat-qr.png" alt="微信二维码" style="width: 200px; height: 200px;">
```

5. 保存文件（Ctrl+S）

---

## 第四步：创建GitHub仓库

1. 登录 GitHub：https://github.com
2. 点击右上角的 "+" 号 → "New repository"
3. 填写信息：
   - Repository name：`teacher-helper`（或你喜欢的名字）
   - Description：`老师的小能手官网`（可选）
   - 选择 **Public**（公开）
   - ✅ 勾选 "Add a README file"
4. 点击 "Create repository"

---

## 第五步：上传文件到GitHub

1. 在新建的仓库页面，点击 "Add file" → "Upload files"
2. 把 `老师的小能手` 文件夹里的所有文件拖进来
   - index.html
   - wechat-qr.png（你准备好的二维码图片）
3. 点击 "Commit changes"

---

## 第六步：开启GitHub Pages（免费部署）

1. 在仓库页面，点击顶部的 "Settings"
2. 左侧菜单找到 "Pages"
3. 在 "Source" 部分：
   - Branch：选择 `main`
   - Folder：选择 `/ (root)`
4. 点击 "Save"
5. 等待1-2分钟

---

## 第七步：访问你的网站

1. 回到仓库主页
2. 右侧找到 "About" 部分
3. 你会看到网站链接，格式类似：
   `https://你的用户名.github.io/teacher-helper/`
4. 点击链接，就能看到你的网站了！🎉

---

## 第八步：分享给老师

把网站链接发给老师朋友：
- 微信发链接
- 发到教师群
- 打印二维码（用草料二维码生成网站链接的二维码）

---

## 常见问题

**Q: 网站打不开？**
A: GitHub Pages 需要1-2分钟生效，等一会再试。

**Q: 怎么修改网站内容？**
A: 在GitHub仓库里直接点击文件，点击铅笔图标编辑，然后保存。

**Q: 怎么绑定自己的域名？**
A: 可以先用GitHub的免费域名，以后有需求再绑定自己的域名。

**Q: 收费怎么实现？**
A: 目前先通过微信手动收款，后续可以接入支付系统。
